# reactjs-text-to-image
React component for convert text to image

## Install

```bash
npm install reactjs-text-to-image
```

## Example

```javascript
import TextToImage from 'reactjs-text-to-image';

...
<TextToImage name="Hello world!" x="0" y="10" />
...

```
